Welcome to torch-train's documentation!
=======================================

torch-train provides a wrapper around the `torch.nn.Module`_ to provide scikit-learn like :py:meth:`fit` and :py:meth:`predict` methods.

.. _`torch.nn.Module`: https://pytorch.org/docs/stable/nn.html#module
